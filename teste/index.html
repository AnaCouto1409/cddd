Antes de tudo: sites não são mágicos

Este site e todos os outros que vivem na internet foram criados por pessoas. Isso mesmo, seres humanos de carne e osso. Então qual o processo para fazer um site aparecer?

A grosso modo, o processo é o seguinte:

Você digita comandos no teclado mandando ele fazer algo.

Computador (super-obediente) executa o que foi pedido.

Com o código na internet mais computadores podem executar estes comandos.

Um PS...

A internet também não é mágica, mas depois entramos nisso. O importante agora é saber que pedindo algo para o computador (na linguagem que ele entende: códigos) conseguimos que ele mostre o conteúdo na tela.

Olha que incrível!

Até hoje você foi uma usuária, que consumia materiais prontos, mas aprendendo a fazer páginas você começa a ditar ordens para o computador! Você começa a dizer para a máquina o que ela pode fazer. Legal né?

Legal, mas onde eu digito os códigos?

Sabe quando você vai escrever um texto e abre o Word ou o Notepad? Essas são ferramentas que ajudam você a escrever textos.
Para escrever códigos, podemos usar ferramentas como Atom, Sublime e Visual Studio Code (e o Notepad também! para programadoras raiz :P).

Atom

Sublime

Visual Studio Code

Existem inúmeras outras ferramentas, mas essas são bem comuns e possuem versões gratuitas, então é só baixar. Mas se você não quer instalar nada ainda no seu computador, existem sites que “simulam” essas ferramentas e você pode escrever códigos por lá também. Aqui alguns exemplos que fazem isso:

Codepen

O CodePen é uma ferramenta em tempo real para escrever códigos, fácil de usar.

JS Fiddle

Com o JSFiddle você também escreve o código e visualiza o resultado na hora; é bastante objetivo.

Habemus

A brasileira Habemus oferece um editor de código na nuvem com a estrutura básica de arquivos.
